Welcome

Hello visitor, this site is still a work in progress so if you're seeing this you must be a classmate or a beta tester. Eitherway I want to say welcome, feel free to check out the Game if you want, it's still finnicky doesn't have save/load and uses a lot of hand-waving, but it's functional. I hope to make games like this for my future CTE courses so the students have a more differentiated and fun learning experience, as 'Gaming' is considered the 4th new addition to the classic three high involvement strategies of Simulation, Roleplaying and Dramatization in classic teaching strategies.
